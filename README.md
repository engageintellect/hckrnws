# hckrnws

## Description

A Hacker News clone built with [Sveltekit](https://kit.svelte.dev), [Tailwind](https://tailwindcss.com), and [DaisyUI](https://daisyui.com).

## Features

- ✅ View top stories
- ✅ View new stories
- ✅ View best stories
- ✅ View ask stories
- ✅ View show stories
- ✅ View job stories
- ✅ View comments
- ✅ View story content
- ✅ View user profiles

## Getting Started

```bash
git clone https://github.com/engageintellect/hckrnws.git
```

```bash
cd hckrnws
```

```bash
pnpm install
```

```bash
pnpm run dev
```
