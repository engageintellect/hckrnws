# hckrnws

## Description

A Hacker News clone built with Sveltekit, Tailwind, and DaisyUI.

## Features

- [x] View top stories
- [x] View new stories
- [x] View best stories
- [x] View ask stories
- [x] View show stories
- [x] View job stories
- [x] View comments
- [x] View story content
- [ ] View user profile (WIP)

## Getting Started

```bash
git clone https://github.com/engageintellect/hckrnws.git
```

```bash
cd hckrnws
```

```bash
pnpm install
```

```bash
pnpm run dev
```
